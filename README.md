


# 歌と日記20260509。

まず、二つの歌をやりました。

中国語の「hong dou」と広東語の「yue ban xiao ye qu」こういう二曲を。

歌ファイルだからgoogle driveにアップロードした、リンクが「utaaddress.txt」というファイルにあります。

そして自分へのツッコミ(笑)。

何このメチャクチャな日本語だよオイ。歌ファイルを作った後、それを再生する時「この人の日本語喋ること流石に...」という感想が出た。

そして声も。なんか女らしい。むしろちょっと子供らしい。長い間にあんまり人間と直接話すことがなかったのせいかも知れない。

明日ってさ、メタバに行きます。

それが一体何の意味がある、分からないです。このブログに日記を書くことや歌をやることみたい。

何も意味のないかも知れない。

最初からただ自分一人でここを更新続いたかも知れないから。

でも、行きます。

あと一週間だけだから。

後悔しませんの為。

相変わらず、一人だけでバカみたい待つことをするか。あるいは何者と出会うか。

ね、バカ。

こういうの流石にカッコ悪いセリフだけど、本当に、最後かも知れないので、来たら嬉しいです。

例え新規アカウントでも。

別に「死なないの為に君を利用するつもり」ではない、この人本当に、もうすぐ食べ物を買うお金もなくなるから、最後で。

そしていきなりだけど、そのみゆきさんの話。

付き合っても、悪くないかも。

最近ってさ、ただ一瞬だけけれど、真面目にそのみゆきさんと付き合うという可能性を考えた。

もうボロボロだから、この私。

ただ本当に、自分のことを大事する人と、恋人になりたい。

そのみゆきさんだけではない、もしある「実はずっと私のことを見守った人物」がいるなら、例えそれがおバカの君ではなくても、その人と話して、恋人となることも、悪くないと思う。

カッコ悪いだろう。

けど、もうボロボロだ。

ただ自分のこと大事する人と出会いたい。そんな人物から慰め言葉を聞きたい。そしてできれば例え日本でもそんな人物のそばに行きたい。

一人だけの旅、本当に怖いだから。

けど、ま、自分でも分かる。

そんな人物、多分、いないかも知れないということよく分かります。

マッチ売りの少女。

という童話読んだことありましたか。

自分は多分、その少女みたい。

うまい晩ご飯を見たけれど、家族を見たけれど。

あくまでただの幻。

マッチの火が消えたら、何もかも、なくなる。

自分を待っているのはただ、咆哮している冬の風、何も見えない真っ黒な夜、それともうすぐやって来るかも知れない死亡。

分かるさ、自覚あるよ、しのやさんが。

この一年、メタバに見た物、ただの美しい幻かも知れない。

最初から、ただいろんな勘違いをして、他人にいろんな迷惑をかけて、そしてバカみたいここで自分しか見る人間がない日記を書くだけ。

夢がそろそろ終わる。ジョーカーさんがそろそろ退場してしまう。

マッチの火が、結局ただの幻。

カッコ悪いだろう、こんなセリフ。

けどさ、ボロボロだったです。

努力をしても、もちろんそんなのはあんまり偉い努力ではないけれど、努力をしても、何ももらえない。

孤独感に襲われたです。

自分を一度殺した、その孤独感に。

明日、メタバに行きます。

もちろんただバカみたい待つだけではない、ワールドに座って、小説を書くかも。

そして万が一、マッチの火だけではない、何かが来るかも知れない。
