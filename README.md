


# ありがとう、おバカのクソ姫様。

拝啓、おバカのクソ姫様へ。

最近ってさ、大変忙しくてこのページを更新する暇がなかったです。

ね、バカ、最近はどう、元気ですか。

何故かお前の名前がマスマス長くなってマスマスひどくなった(笑)けれど、ま、そんなのどうでも良いだ。

ところで一体いつから「クソ姫様」ていうのを付いたんだろう。あんまり覚えなかった。台湾映画を見た頃のことだったかな、確かお前に「お姫様になることがダメ」言いあげたかどうか。

ともかく、ようやくちょっと暇を取ったので、久しぶりにここを更新します。

一週間5日で働く、深夜3時まで勤務することもよくある。これが最近の状態だった。

しかも、深夜まで勤務することが仕事の中の一番ラクなことだと思う。今やってるこの飲食店の仕事がこんなひどい物です。

体にとってもちろん、こんな仕事があんまり良いこととは言えない。いつもクリーニングの勤務をするのせいかもが、最近なんか指がちょっと腫れたみたい。

でも、ま、給料で買った食べ物をメチャ食べた時、どんなひどいことも忘れられると思う(笑)。

甘い物を食べたら、肉を食べたら、そしてエロ写真を見たら、基本的に満足です。しのやさんがこんな満足しやすい人物です。

Nのために、ていうドラマ見たことありますか。

「ネタバレ」のせいで「ネタバレ」たけれど、「ネタバレ」のおかげで助けられて、全部やり直すと言いながら大学をやめて、レストランの仕事をやって、ある日お店のゴミを片付ける時ふと夕日を見た成瀬君。何故かそれを思い出した。

やり直す、か。

ドラマと違う、リアルにはやり直すということがそんな甘いことではない。

日本にいる時、仕事も大変忙しかったけれど、少なくとも体力的な勤務をする必要がないし、チームリーダーだからイジメられることもないし、しかも何より給料がピカピカで、もちろんそこまでピカピカではないけれど少なくとも一ヶ月に15万円ぐらいの貯金をできて、食べたい物もやりたいゲームも何もかもすぐ買うことが可能。

そして今、毎日深夜まで働いても一ヶ月3万円ぐらいしか稼げない、貯金なんともちろんほぼできない。思い出したら、かつて日本での生活がまるで夢みたい。

けど、いや、少なくとも生きている。

大変ひどいけれど、生きている。

まだ肉を食べることが可能、エロ写真を見ることも(笑)。


そう考えると、そこまで納得できないとは言えないな。

居場所がない。何もできない。

それがこの飲食店の仕事をやった後気が付いたことだったです。

この私、一体なにをできるだろう。

クリーニング程度の仕事でも、うまくできない。毎日深夜まで働くけれど、実はそんな重い仕事ではない。ただこの自分がこんな仕事をうまくやれないだけ。

だからさ、時々こういう質問を自分に押す。

この自分、一体何をやれるだろう、と。

日本語も見て通り、今でもこの中途半端のまま。小説の書くことも。理想があるけれど、ただ「ある」だけ。

しかも三十歳。

時々、元モデルさんの南先輩みたいに、自分一体なにをやれるだろうと、自分に質問することがある。

けど、多分、この自分でも、何かをやれる。このことだけを、ずっと信じている。

少なくともまだ生きているだろう。

なら見つけてしよう、未来とか。

生きてることが何よりだから。

そしてさ、バカ。

ありがとう、という言葉を、もう一度、お前に言う。

もしお前と出会わなかったら、多分、今の私もいないから。

もちろん、君に恨みが全然ない、というのは言えない(笑)。

君というクソお姫様からいろんな痛い目をもらったから。

今でも「もしこのおバカさんとリアルに出会ったら一番やりたいことが何」と言われたら「このクソ野郎のバカ頭を渾身の一撃で殴りたい」と、君への怒りがこの程度な物です(笑)。


でも、君にずっと、感謝という気持ちがあります。

そしてさ、いきなりだけど、例の武侠小説のヒロイン(？)、宮月という人物の話をします。

人を恋する能力が、失った。

それがその宮月のことです。

一度で丐幇幇主の洪玄清のこと好きなったけれど、一度で暁風のこと好きなったけれど、よく考えたら、それが多分「好き」とは言えない。

本気で、真面目に恋をすることは多分、一回だけ。

元の時代の、その「姉の彼氏」を間違って好きなったことだけ。

もちろんその少年がそこまで優秀な人物ではない、顔も性格も何もかもただの「どこにもいる」さんけれど。

タイミング、そのヤバい物のせいだから。

ものすごく好きになって、相手の存在を魂まで焼き付けたから。

その焼き付けた物を無理やり取ったら、もちろん生きることがまだできるけれど、多分、何かを失ったかも知れない。

人を恋する勇気とか。

だから小説の第三冊に、間違って既婚者の暁風先生に好きなったけれど、暁風夫人のおかげでその恋情を忘れて、暁風先生と良い友達になった。

最初から、宮月が暁風への気持ちが、本物の恋ではないから。

好きだった人と良い友達になる。こういうふざける話が多分、小説にあるだけ。


恋情と友情と、水と油みたい絶対に混じり合うことできないから。

だから、宮月という小娘が洪玄清とも、暁風とも、純粋な友達になれる。

人を恋する能力、というのは本の昔から、元の時代に既に失ったから。

別にその「姉の彼氏」を忘れられないではない、ただその叶わない恋情のせいで、心に傷を残って、穴を残っただけだ。

「哀莫大于心死」。

「笑傲江湖」という武侠小説に「瀟湘夜雨」という人物がいる。自分の大好きな武侠小説の人物。興味があればその武侠小説を読んでね、確か日本語のバージョンもあるみたい、この私の劣作と違う、本格的な武侠小説で、この自分の大好物の一作。

自分の作品の話を。

自分のその武侠小説の主人公がもちろん暁風けれど、言うなら暁風だけではない。小説を執筆する宮月も、丐幇幇主の洪玄清も、そして前の間に言った段も、白芷も、あとお前に言ったことのない悪役の万毒教の教主も、どっちでもこの作品の主人公と言えます。

そしてここだけの話だけど、実は自分は勝手に宮月の声優を喜多村さんに想定した(笑)。原型の一つがSAOのクソ黒の剣士野郎の妹だから。

ともかく、今でもこの作品の完成を目指しています。

もちろん、その宮月の原型の一部がこの作者の私けれど、ただの「創作」だけ。

もし、「正しい人」と出会ったら、新しい物語を始めることもできると思う。

でも、ま、少なくともしのやさんが今でも独身だな(笑)。
