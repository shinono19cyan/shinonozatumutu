だから、ここでちょっと日記を書く。

一ヶ月。

あと一ヶ月だけ。

元々さ、やる気満々で家賃の相談を行きました。何より今の住んでる場所の家賃が安い、ここに住んでたこの一年ずっと家賃をきちんと支払っていたから、ただ「一ヶ月の滞納」なんて、そこまで難しいことではないかなと思ったけれど。

お金を支払って、あるいは出てください。と言われた。

そう。

なんか、大人への道にもう一歩をした気がする。

大人の世界が厳しいということよく知ってるけれど、そこまで厳しいなんて、流石にちょっと痛い。

最後では何個の口座に残ってる本の少しだけの預金を集めて、ギリギリと一ヶ月分の数字をできた。

残ってるお金、3000日本円程度の物。

そして約二週間分の食べ物。

そう、これだけ。

しのやさんの人生がマスマス小説っぽいになったみたい。

笑っても良いよ。

良かったな、少なくとも笑いをもらえるなんて。

分からない。

これからどうする、どうになる、全然分からない。

もしかしたら、前の間に自分の手で命を終わらせるこそ、正しい選択肢かも知れない。

生きる、という選択を選んだうえ、今その選択肢の重さを耐えているということだけを知ってた。

episode: 鈍感とプライド。

一つの作品を君にオススメします。

傾物語。

物語シリーズの一冊。

ある「可能性」のこと。

もし、化物語の最後のエピソード、猫のエピソードに、阿良々木が決戦の時に忍ちゃんの「行方不明の謎」を解決しなかった、という「並行世界」の物語となります。

世界が傾いた。

ちょっとネタバレかも知れないけれど、世界が傾いたんです。タイトルの傾、多分こういう意味と思う。

一人の鈍感な男が、一人のプライドのせいで本心を言えない女の子の謎を解決しなかったのおかげで、世界その物が傾いた。

鈍感。

プライド。

小説なんてあくまでフィクションだけど、リアルの「残念だった」物語には多分、一部がこういう感じ。

鈍感とプライド。

謎を解決できない鈍感さんと、謎を相手に押したプライドさん。

こういう話は昔にもやったことがあるけれど。

君の意味不明な頭を、マジで、一発で殴りたいこのチクショウ(笑)。

もし、その翼さんが本当に君なら。

その人がやった何かをアピールしたいポスト、アバター名の写真、それを見たことが本の後のことだった。多分四月の頃のことだった。

翼、という名前が描いてる写真。


正直さ、今でも、あの人物の正体を知りませんです。

ただ、その名前と、あの日の対話の内容で、それがもしかしたら君かな、という推測があるだけ。

もしそうなら。

信じてください、君を殺さないですから、君の頭を一発殴っても良いですかバカ小娘(笑)。

いや分からないさ、お嬢さん。

分かるわけがないだろう。

人間ってさ、名探偵ではないよ。

何でも知ってる名探偵なんて、あれ多分、本当にただのフィクションと思う。リアルにはそんな人物がいないと思う。マジで。

時々さ、もしあの頃私がその写真を見たら、物語がどうになるだろうと、自分に問うことがあります。あんまり意味がない反省だけど、時々そう反省することがありました。

でも、ま、結局にはこの「世界」の私がそれを気付かない、最後にも君と話せない、そして余命一ヶ月(笑)になった。これこそこの世界の物語。残念の物語だった。

その傾物語の並行世界に、悪いのは阿良々木か。あるいは忍ちゃんか。

どっちでも、悪いと思う。

鈍感さんとプライドさん。
