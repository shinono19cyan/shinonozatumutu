


# 混乱だけど。

結局しのやさんに正面的の物をあげる人、一人もないみたい(笑)。

月曜のイベント、ただ一人だけ(笑)でデジモンのアニメを楽しそうにみただけ。最後につい同好さんが来たけれど、ただ簡単に話し合えっただけ。多分ただのメタバの新人さん。

昨日も、メタバに行ったけれど、一人だけで時間を過ぎた。

ま、いつもこんな感じだけど。

多分しのやさんに「何か」をあげる人、一人もいないということかも知れない。

一方、界王のところにいる悟空達。ではなくて(笑)。

何故この人いつでも笑えるの。

笑える場合ではないだろう。

この物語の先、一体どうなるだろう、だろう。









# 冷静して。

冷静にしてね、バカ。

最近ってさ、もうすぐ「終わるの日」かも知れないから、頭がちょっとおかしくなるみたい。終わるの日という言葉をまるで他人のことみたいに言うなんて、自分は今こんなちょっとおかしい状態です。

それ以上このブログを読むな。大事に生きてください。

と、カッコ良くて言って伝説の優しい人になりたいだけど。むしろそれそこ「正しいこと」というのよく分かるけれど。

分からないさ。

今の気持ち、分からないさ。

知ってるのはただ、今でも、君に会えないということを想像するだけでも、ものすごく悲しくなる。

そして、やっぱり、君が元気で生きたら私にはそれだけで十分、ということだけ。知ってるのはこれだけ。

君のポスト、メタバの写真で君の今週の中華ドレス姿を見た後君のポストを見た。

気のせいかも知れないが、最近のポストが消去されたみたい。何より元々何日前の今週のイベントお知らせがあるわけなのに。

バカなことしませんのように。

分かる、今の君が混乱。私も。けど、例え返事できなくても、バカなことしませんのように。

元気で、生きてください。

前の間書いた「手紙ファイル」、見たことがないならそれを読んでね。

大事にしてくださいよ。
