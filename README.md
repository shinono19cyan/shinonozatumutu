


# 新作の予告。

タイトル、「バカ探偵しのやと蛇と猫」。

小説と言ったけれど、ただここで発表します。

何よりしのやさん自分の話だから。

一人の、ロビーに出会ったデジモンが好き人の物語。

今でも、その人の正体を確信できない。もし間違ったら、どっちにも傷を付くです。

その人多分、私に会うことが「一回だけではない」。

毎回、違うアカウントだけど。

容姿を変われる連続殺人鬼みたいな人物。

しのやという名前の物語、ただ本の少しだけですから。最後までその人の正体を分からないかも知れない。

その毎回、しのやが苦しんでる時、違うアカウントでしのやに慰めてあげて、嘘ばっかりだった犯人の正体。

だから、ただここで、その人と出会うこと、そしてしのやの推理をしますだけ。

良かったらその人の自首をもらいたいだけどな(笑)。

しのやさんの命、本当に、あと少しだけだから。

一体どっちだろう。

分からない。

コイツもドイツも、嘘ばっかりで本心を言えない。

しのやさんってさ、ただ一人だけの女の子、自分に本心を言える女の子と恋人になりたいだけです。なにこの物語、登場人物があんまり多すぎではないか。けどどうして自分の本心を正々堂々と言える人が一人もないの。

しのやさんはただ、一人だけの女の子と恋して、別の全ての女性を忘れたいだけです。

あと土曜日、イベントを開催するかも。

予告がここまで。
